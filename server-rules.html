QLDRP FiveM Discord Rules

  • Rule 1: Be respectful, civil, and welcoming.
  • Rule 2: No inappropriate or unsafe content.
  • Rule 3: Do not misuse or spam in any of the channels.
  • Rule 4: Do not join the server to promote your content.
  • Rule 5: Any content that is NSFW is not allowed under any circumstances.
  • Rule 6: The primary language of this server is English.
  • Rule 7: Discord names and avatars must be appropriate.
  • Rule 8: Spamming in any form is not allowed.
  • Rule 9: Do not attempt to bypass any blocked words.
  • Rule 10: Don’t ping without legitimate reasoning behind them.
  • Rule 11: No Discord server invite links or codes.
  • Rule 12: Do not advertise without permission.
  • Rule 13: Set your Discord nickname to you FiveM nickname
  • Rule 14: Do not post doxxing material.
  • Rule 15: Only request a SSU once per 6hrs (this is server-wide), this must be done via the /server request command.
